# 任务中心群聊创建功能测试指南

## 功能概述
简易模式AI助手中心的任务版块已重新设计，现在用户点击任务模块时会：
1. 弹出问题输入对话框，让用户直接提出问题
2. 自动检测已配置API的AI服务，创建专家群聊
3. 发送专家身份提示词，引导AI以专家身份讨论
4. 将用户问题作为用户消息发送到群聊中
5. 立即触发所有可用AI自动回复，无需其他操作
