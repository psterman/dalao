## 🏷️ 左侧标签详细设计

### 标签项布局（item_collection_type.xml）

```
┌─────────────────────┐
│  🤖                 │  ← 图标（24dp × 24dp）
│  AI回复收藏          │  ← 文字（14sp）
│  (12)               │  ← 数量（11sp，灰色）
└─────────────────────┘
```

**选中状态**：
- 背景色：`#E3F2FD`（浅蓝色）
- 文字颜色：`#2196F3`（蓝色）
- 左侧指示条：4dp宽，蓝色

**未选中状态**：
- 背景色：透明
- 文字颜色：`#333333`（深灰色）

### 卡片元素尺寸

- **图标**：32dp × 32dp，左侧，圆角8dp
- **标题**：16sp，粗体，最多2行

- **内容预览**：14sp，最多3行，行间距1.2
- **标签**：10sp，圆角标签，内边距4dp×8dp
- **底部信息**：12sp，图标+文字

## 🔍 筛选栏详细设计

### 筛选栏布局（layout_filter_bar.xml）

```
┌─────────────────────────────────────────────────────┐
│  [全部] [优先级:高] [未完成] [最近7天]  [搜索框🔍]   │
└─────────────────────────────────────────────────────┘
```

**筛选按钮样式**：
- 高度：36dp
- 内边距：12dp × 8dp
- 圆角：18dp（胶囊形状）
- 未选中：白色背景，灰色文字，1dp灰色边框
- 选中：蓝色背景，白色文字

**搜索框样式**：
- 高度：36dp
- 圆角：18dp
- 背景：浅灰色（#F5F5F5）
- 图标：左侧搜索图标，右侧清除按钮

## 📊 网格视图模式

### 2列网格布局

```
┌──────────────┬──────────────┐
│  [图标]      │  [图标]      │
│  标题1       │  标题2       │
│  来源 · 时间 │  来源 · 时间 │
│  [标签]      │  [标签]      │
└──────────────┴──────────────┘
┌──────────────┬──────────────┐
│  [图标]      │  [图标]      │
│  标题3       │  标题4       │
│  来源 · 时间 │  来源 · 时间 │
│  [标签]      │  [标签]      │
└──────────────┴──────────────┘
```

**网格项尺寸**：
- 宽度：(屏幕宽度 - 边距) / 2 - 间距
- 高度：自适应，最小120dp

## 📐 间距规范

- **卡片间距**：8dp（垂直），16dp（水平）
- **卡片内边距**：12dp
- **元素间距**：8dp（小），16dp（中），24dp（大）
- **列表边距**：16dp（左右），8dp（上下）

## 🎭 交互动画

### 卡片点击
- 缩放：0.98倍
- 时长：100ms

### 标签切换
- 淡入淡出：200ms
- 滑动：300ms

### 对话框弹出
- 从底部滑入：300ms
- 背景渐显：200ms

---

## 📱 响应式设计

### 小屏幕（< 600dp宽度）
- 左侧标签：固定100dp
- 右侧列表：剩余宽度
- 卡片：单列，全宽

### 大屏幕（≥ 600dp宽度）
- 左侧标签：120dp
- 右侧列表：剩余宽度
- 卡片：可考虑2列（如果选择网格模式）
